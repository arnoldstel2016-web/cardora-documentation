
# CARDORA Google Play Data Safety

**Status:** In Progress

---

# Overview

This document describes the data collection, processing, storage, and security practices of CARDORA for Google Play Data Safety compliance.

All declarations submitted to Google Play must accurately reflect the actual behavior of the application.

---

# Data Collection

## Personal Information

Collected:

* Display Name
* Email Address
* User ID

Purpose:

* Account creation
* User authentication
* Account management

---

## Financial Information

Collected:

* Purchase History

Purpose:

* Membership activation
* Purchase verification
* Subscription management

Payment information is processed by Google Play and is not stored by CARDORA.

---

## Photos and Media

Collected:

* Photos selected by the user
* Camera images captured by the user

Purpose:

* Trading card recognition
* Card scanning

Images are processed only for card recognition.

---

## User Content

Collected:

* Collection
* Wishlist
* Favorites
* Portfolio

Purpose:

* Synchronization
* User experience
* Cloud backup

---

## Device Information

Collected:

* Device identifiers (where required)
* Crash diagnostics

Purpose:

* Application stability
* Bug fixing
* Performance improvements

---

# Third-Party Services

CARDORA currently uses:

* Supabase
* Google Play Billing
* Google AdMob
* Google Sign-In
* Pokémon TCG API

Each service processes data according to its own privacy policy.

---

# Data Sharing

CARDORA does not sell personal data.

Data is shared only when required to provide application functionality through trusted service providers.

---

# Data Security

CARDORA protects user data using:

* HTTPS encryption
* Secure authentication
* Row Level Security (RLS)
* Secure cloud storage
* Access controls

---

# User Controls

Users may:

* Access their account data
* Update profile information
* Delete their account
* Request data removal

---

# Account Deletion

Users can permanently delete their CARDORA account.

Upon deletion, personal data associated with the account is removed in accordance with the Privacy Policy.
