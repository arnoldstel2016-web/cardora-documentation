# CARDORA Brand Guidelines

**Status:** Active

---

# Overview

The CARDORA brand represents professionalism, trust, innovation, and a passion for trading card collecting.

These guidelines ensure that the CARDORA identity remains consistent across all platforms, applications, websites, documentation, and marketing materials.

---

# Brand Identity

CARDORA is designed to be:

* Modern
* Clean
* Premium
* Community-focused
* Collector-friendly

Every visual element should reflect these values.

---

# Logo

The official CARDORA logo should always be used without distortion.

The logo must:

* Maintain its original proportions.
* Remain clearly visible.
* Have sufficient spacing around it.
* Never be stretched or compressed.
* Never be rotated.
* Never be recreated manually.

---

# Typography

Documentation and official communication should use clean and readable typography.

Recommended style:

* Clear headings
* Consistent spacing
* Professional formatting
* High readability

---

# Visual Style

CARDORA uses a modern interface with:

* Rounded elements
* Clean layouts
* Premium visual appearance
* Consistent spacing
* Minimalistic design

---

# Icons

Icons should be:

* Simple
* Consistent
* Easy to recognize
* Used only where they improve usability

---

# Images

Marketing images should:

* Be high quality
* Represent actual application features
* Match the CARDORA visual identity
* Avoid excessive visual effects

---

# Certificates

Official certificates issued by CARDORA must:

* Use the official branding
* Display accurate information
* Include sequential numbering where applicable
* Maintain a premium appearance

---

# Brand Consistency

All CARDORA materials should maintain a consistent appearance across:

* Mobile applications
* Documentation
* Website
* Social media
* Marketing materials
* Presentations
