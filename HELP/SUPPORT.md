# CARDORA Support

**Status:** Active

---

# Overview

CARDORA Support provides assistance for technical issues, account management, memberships, purchases, and general questions.

Our goal is to help users resolve issues quickly while providing a reliable and professional support experience.

---

# Support Topics

Users may request assistance with:

* Account creation
* Sign in
* Password reset
* Google Sign-In
* Scanner issues
* Collection management
* Memberships
* Purchases
* Application errors

---

# Membership Support

Support is available for:

* Founder Membership
* Early Supporter Membership
* Pro Membership
* Purchase verification
* Restore purchases

---

# Technical Support

Technical assistance includes:

* Application crashes
* Performance issues
* Synchronization problems
* Scanner recognition issues
* Network connectivity
* Update problems

---

# Bug Reports

When reporting a bug, users should include:

* Device model
* Android or iOS version
* CARDORA version
* Description of the issue
* Steps to reproduce
* Screenshots (if available)

---

# Feature Requests

Suggestions for improving CARDORA are welcome.

Feature requests are reviewed for future releases but cannot be guaranteed.

---
